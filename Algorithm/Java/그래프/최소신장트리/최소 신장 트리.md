# 최소 신장 트리

> MST, Minimum Sanning Tree

#### 그래프에서 최소 비용 문제

1. **모든 정점을 연결하는 간선들의 가중치의 합이 최소가 되는 트리 - 최소신장트리**

2. 두 정점 사이의 최소 비용의 경로 찾기 - 최단 경로

<br>

#### 신장 트리

* n개의 정점으로 이루어진 무향 그래프에서 n개의 정점과 n-1개의 간선으로 이루어진 트리

<br>

#### 최소 신장 트리

* 무향 가중치 그래프에서 신장 트리를 구성하는 간선들의 가중치의  합이 최소인 신장 트리

<br>
